## ✨ Features

- 📝 HTML, CSS, and JavaScript input panes
- ⚡ Live preview of output as you type
- 🧹 Clear/reset buttons
- 💾 Uses `localStorage` to optionally retain code (if implemented)
- 🖥️ Responsive and minimal UI

---

## 🧰 Tech Stack

- **HTML** – Structure
- **CSS** – Styling
- **JavaScript** – Editor logic, real-time preview
